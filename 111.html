📘 Notes de lecture : « Le rose Barbie serait la plus ancienne couleur jamais créée »

🟣 7. Les enfants et le rose

Texte : Les enfants nés dans les années 1800 portaient tous du blanc. C’était plus pratique pour les laver et, de toute façon, on pensait que les enfants ne voyaient pas les couleurs.

📌 Vocabulaire :
  • puériculture : n.f., soins pour enfants en bas âge
  • affirmer : déclarer, dire avec certitude
📖 Traduction : 19世纪出生的孩子都穿白衣服，便于清洗，而且当时人们认为孩子看不见颜色。
🔍 Analyse : on pensait que... 是一个常见表达，用于表示“人们认为……”。

🟣 8. La féminisation du rose

Texte : Ce n’est que dans les années 1950... qu’elle devienne le symbole d’un consumérisme domestique et féminin.

📌 Vocabulaire :
  • consumérisme : n.m., 消费主义
  • domestique : adj., 家庭的
📖 Traduction : 直到20世纪50年代，玫红色才真正成为一种象征女性家庭消费主义的颜色。
🌍 Culture : 二战后，家庭用品中大量使用粉色，如厨房、浴室，代表“理想主妇”的柔美形象。

🟣 9. Barbie et le féminisme

Texte : En 1959, Barbie voit le jour... Aujourd’hui, elle est réinterprétée dans une perspective féministe.

📌 Vocabulaire :
  • voir le jour : 诞生、出现
  • adulé : 被热烈崇拜的
  • objectification : n.f., 物化
  • réinterpréter : 重新解读
📖 Traduction : 1959年芭比诞生。她曾是女性魅力的象征，也因女性物化而被批评。今天，她被重新解读为女权象征。

🟣 10. Une couleur queer ?

Texte : Aujourd’hui, le rose est souvent associé aux luttes LGBTQIA+, de la pink triangle au rainbow flag.

📌 Vocabulaire :
  • lutte : n.f., 斗争，抗争
  • pink triangle : 粉红三角（LGBT 历史标志）
  • rainbow flag : 彩虹旗
🌍 Culture : 粉红三角源自纳粹时期对同性恋的迫害标志，后来被 LGBTQIA+ 社群夺回，作为骄傲的象征。
📖 Traduction : 如今，玫红色被视为 LGBTQIA+ 社群抗争和认同的象征。

🟣 11. Conclusion

Texte : Une couleur n’est pas un genre. Et parfois, un rose n’est qu’un rose.

📖 Traduction : 颜色并不等于性别。有时候，玫红只是玫红。
